# 战士技能两份文档去重合并分析

> 基准文档：`warrior_tactical_skill_system_200.md`（旧版，基于 D&D 九剑风格）
> 对照文档：`warrior_tactical_skill_system_new_200.md`（新版，声称避开旧版名称）
> 去重结果：新版 200→167（移除 33 个功能重复）
> 朝向适配：367→352（移除 15 个依赖单位朝向系统的技能）
> 引擎能力对齐：吸血/反伤/召唤/坐骑**保留**（设计预留，暂不要求引擎支持）

---

## 一、朝向机制移除（15 个）

游戏当前**没有单位朝向系统**，无法判别正面/背面/侧面。以下技能依赖朝向，予以移除：

### 旧版（7 个）
| 编号 | 技能名 | 依赖的朝向机制 |
|------|--------|--------------|
| #015 | 战术转身 | 转向并横移 1 格 |
| #125 | 背刺 | 从背后攻击时伤害变 200% |
| #127 | 虚步 | 移动找背击位置 |
| #134 | 诡影连击 | 若从背后攻击，可追加第三击 |
| #139 | 十步无声 | 背击命中 +20% |
| #140 | 万影绝杀 | 若背击，追加两次 80% 影刃 |
| #177 | 烈阳一闪 | 若从正面攻击，目标无法反击 |

### 新版（8 个）
| 编号 | 技能名 | 依赖的朝向机制 |
|------|--------|--------------|
| 铁潮#1 | 肩甲撞击 | 背击目标时附加失衡 |
| 星刃#2 | 月弧侧斩 | 从侧面攻击时造成 150% |
| 星刃#11 | 无声切脉 | 背击时造成 150% |
| 盾环#7 | 肩盾破位 | 将目标朝向改为背对一名随机友军 |
| 双刃#10 | 裂影二择 | 若目标朝向你，则额外造成一次 50% |
| 军势#9 | 侧翼暗号 | 目标下次从侧面攻击时伤害 +20% |
| 影手#125 | 背刺 | （旧版同名，已在旧版移除列中） |
| 落日#146 | 转身投 | 使目标转向背对自己 |

## 二、引擎暂不支持但保留的技能机制

以下机制当前 `CombatEffectDef` 不支持，但作为**设计预留**保留在文档中：

| 机制 | 涉及技能数 | 说明 |
|------|----------|------|
| 吸血 (Lifesteal) | ~3 个 | 如旧#106 嗜血、新龙血(已移除的#167) |
| 反伤/反弹 (Reflect) | ~4 个 | 如旧#071 誓魂护体、旧#090 大地反弹、新盾环#6 盾面反震 |
| 召唤物 (Summons) | ~3 个 | 如旧#092 石柱突起、旧#129 幻影诱饵、新枪林(已移除的#40) |
| 坐骑/反骑兵 (Mounts) | ~1 个 | 如新军势#4 拒马口令 |
| 技能沉默 (Silence) | ~2 个 | 如旧#111 裂喉打击、新枪林#14 白线封魂 |
