# 战士战棋技能系统 200 技能版

> 设计目标：在之前 80 个战棋化战士技能的基础上，扩展成一套可用于 SRPG / 战棋游戏的 200 技能池。
> 
> 参考思路：借鉴 DND 3.5《Tome of Battle: The Book of Nine Swords》的“武技流派 / Maneuver / Stance”设计方法，但不照搬原技能文本与数值。这里全部技能均为原创战棋规则。

---

## 0. 统一字段说明

- **R**：射程。R1 表示相邻 1 格，R3 表示 3 格内。
- **范围**：单体、直线、扇形、小圆、十字、自身周围等。
- **Hit**：相对基础命中修正。例如 Hit+5 表示命中率 +5%。
- **DMG**：攻击倍率。例如 DMG150% 表示造成 1.5 倍武器伤害。
- **消耗/CD**：体力、斗气消耗与冷却回合。
- **AI逻辑**：敌方或自动战斗时的使用条件。

---

## 1. 流派结构

本版本采用 10 个流派，每个流派 20 个技能，共 200 个：

1. 钢心流派：武器大师、正面破阵。
2. 明镜心流派：专注、命中、预判。
3. 白鸦军略：指挥、战旗、团队协同。
4. 誓魂守护：坦克、嘲讽、恢复、防护。
5. 岩龙流派：地形、重压、破甲、击倒。
6. 虎爪流派：跳跃、流血、多段、狂暴。
7. 影手流派：背刺、残影、暗袭、削弱。
8. 落日柔术：借力、投摔、换位、反制。
9. 炎风流派：斗气远程、燃烧、疾风位移。
10. 真龙斗气：斗气、龙魂、真龙斩、终极技。

---

## 2. 通用 AI 决策优先级

1. **能击杀则优先击杀**。
2. **能控制高威胁目标则优先控制**。
3. **自身危险时先防御 / 回复 / 后撤**。
4. **命中多个目标时优先 AOE**。
5. **无法攻击时使用位移接敌或战术站位**。
6. **斗气 ≥ 80 时寻找终极技窗口**。

目标评分建议：

```text
目标评分 = 击杀价值 + 威胁值 + 命中收益 + 地形收益 - 风险
```

## 3. 技能树落地建议

### 3.1 角色最多装备技能
建议一个角色最多装备：

- 普攻 / 基础武技：1 个
- 主动技能：4 个
- 反应技能：1 个
- 姿态 / 状态技能：1 个
- 终极技：1 个

这样不会因为 200 技能导致战斗 UI 过载。

### 3.2 技能解锁层级
建议每个流派 20 个技能拆成 5 层：

- Tier 1：基础技能，角色 1-3 级可学。
- Tier 2：流派核心，角色 4-7 级可学。
- Tier 3：构筑技能，角色 8-12 级可学。
- Tier 4：高级联动，角色 13-17 级可学。
- Tier 5：终极技能，角色 18+ 可学。

### 3.3 武器适配

- 长枪：钢心、岩龙、真龙斗气。
- 大剑：钢心、炎风、真龙斗气。
- 剑盾：誓魂、白鸦、落日。
- 双持：虎爪、影手、炎风。
- 巨斧：虎爪、岩龙、钢心。
- 战锤：岩龙、誓魂、白鸦。

## 4. 平衡建议

### 4.1 控制递减
同一目标连续受到硬控时：

- 第一次：完整持续。
- 第二次：持续减半。
- 第三次：免疫硬控 1 回合。

### 4.2 位移限制
- 穿越型位移 CD 不低于 3。
- 攻击后再移动的技能伤害应比纯输出技能低 10%-25%。
- 不能穿墙，除非技能特别注明。

### 4.3 斗气限制
建议获取：

- 普攻命中：+8 斗气。
- 技能命中：+10 斗气。
- 受击：+5 斗气。
- 击杀：+15 斗气。
- 完美防御 / 反击成功：+10 斗气。

### 4.4 连击限制
战棋中不建议真正无限连。建议采用：

- 普通连击最多 2 次追击。
- 高阶终极可做到 3-4 次，但必须消耗斗气或需要前置层数。
- 每次追击命中率递减 5%-15%。

## 5. Demo 首发建议

如果先做 Demo，不建议一次开放 200 技能。推荐先开放 30 个：

- 钢心：重击、横扫、裂甲斩、破军斩、钢心裁决。
- 明镜：集中精神、破绽读取、心眼反制、精准连击。
- 白鸦：战吼、战旗、战术指令、轮换阵位。
- 誓魂：格挡、护盾墙、挑衅、战斗回复。
- 岩龙：震地、山崩推击、岩肤架势。
- 虎爪：跳斩、撕裂斩、狂暴、嗜血。
- 影手：影袭、背刺、烟幕撤离。
- 落日：后撤步、借力摔。
- 炎风：剑气斩、焰弧横扫。
- 真龙：斗气斩、斗气觉醒、真龙斩。
