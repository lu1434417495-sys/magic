# 全新战士战棋技能系统：200个新技能
> 版本：New-200 Tactical Warrior Skills  
> 目标：在之前80个战棋化技能之外，重新设计200个不同名称、不同定位、不同战棋规则的战士技能。  
> 说明：参考“九剑”式的流派化设计思路，但不照搬原有名称或规则；本文件所有技能均为原创战棋规则。
---
## 设计约束
- 尽量避开旧技能名：重击、横扫、穿刺、旋风斩、裂甲斩、断头斩、连斩、剑气斩、破军斩、真龙斩、冲锋、跳斩、后撤步、影袭、格挡、护盾墙、盾击、挑衅、战吼、狂暴、斗气觉醒等旧技能名。
- 所有技能都按战棋逻辑设计：格子、射程、范围、回合、冷却、状态、AI使用条件。
- 不设计实时动作判定，不设计真正无限连。
- 每个流派20个技能，共10个流派，总计200个。
---
## 统一字段
| 字段 | 说明 |
|---|---|
| 类型 | 输出、控制、位移、防御、状态、斗气、指挥、终式等 |
| 射程 | 战棋格距离，`自身`表示以自身为中心 |
| 范围 | 单体、线、扇形、圆形、十字、路径、友军等 |
| 消耗 | 体力、斗气、生命代价或无 |
| CD | 冷却回合 |
| 效果 | 技能实际规则 |
| AI逻辑 | 敌方AI何时优先使用 |
---
## 流派总览
| 流派 | 主题 |
|---|---|

| 裂岩重兵 | 重武器、破阵、地形互动 |

## AI通用决策模板

### 1. 技能评分

```text
技能评分 = 伤害收益 + 控制收益 + 击杀收益 + 站位收益 + 团队收益 - 风险成本 - 资源成本
```

### 2. AI优先级

1. 可击杀关键目标时，优先使用斩杀或高倍率技能。
2. 敌方核心单位将行动时，优先使用行动延后、缴械、沉默、减命中类技能。
3. 自身低血或将被集火时，优先使用防御、位移、替伤、回复技能。
4. 目标聚集时，优先使用区域技能；但若会误伤重要友军，则降低评分。
5. 斗气高于70时，保留给终式；除非当前技能能形成击杀或救命。
6. 指挥类AI优先保证阵型，而不是盲目输出。

### 3. 控制递减建议

- 同一单位连续受到硬控时，第二次持续减半，第三次免疫1回合。
- Boss或精英不完全免控，而是将硬控转化为行动顺序后移、命中下降或技能消耗上升。
- 位移类强制移动不能穿墙、不能穿越不可通行格，除非技能明确说明。

### 4. 资源建议

- 普通攻击命中：+6斗气。
- 技能命中：+3斗气。
- 受击：+4斗气。
- 击杀：+12斗气。
- 终式消耗建议：斗气65~85。
