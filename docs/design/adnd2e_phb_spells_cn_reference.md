> 
> 我没有逐字翻译整本规则书原文；这里提供的是 **字段化整理 + 中文效果说明 + 实用速查结构**，适合查表、建库、做 Godot 数据表、团务速查。

## 使用说明
- 范围：PHB 核心法术。
- 结构：按 **法师（Wizard）1–9环**、**牧师（Priest）1–7环** 分组。
- 本文件保留英文法术名，方便对照原书、做数据库主键与 VTT/游戏项目录入。
- 每条条目包含：环级、学派、射程、持续时间、施法时间、范围、豁免、中文效果摘要。
- **中文效果摘要是用途向的规则说明，不是规则书正文的逐字翻译。**
