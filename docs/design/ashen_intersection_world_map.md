# 灰烬交界特殊地图设计

更新日期：`2026-04-13`

说明：
- 本文档用于设计一张 `100 x 100` 的特殊世界地图，用作“大地图主世界”和“黑魂风格灰烬世界残片”交汇的单独预设。
- 地图风格是 `黑魂系氛围`，但地名、据点和交汇机制保持项目内原创设定，避免直接复制现成关卡。
- 技能学习与服务奖励统一复用当前仓库的 `settlement action -> pending character reward -> knowledge_unlock / skill_unlock / skill_mastery` 设计方向。
- 技能效果不直接照搬黑魂原始规则，而是改写成当前项目已存在或已规划的 `single / line / cone / radius / self / ground / cast_variants / status / passive` 技能体系。

---

## 一、地图概念

- 地图名称：`灰烬交界`
- 地图规模：`100 x 100`
- 地图定位：主世界中的一处“界火裂隙”失稳后，把一块灰烬王朝废土拖入当前世界，形成一张可独立探索的异界特区。
- 游玩气质：
  - 中央据点安全，但外围层层压迫。
  - 大路看似直通，实际通过断桥、墓园、教区、书库、回廊形成多重环路。
  - 据点数量不多，但每个据点都承担明确功能：休整、锻造、战技、祷告、咒术、追忆、誓约。
- 当前系统约束：
  - 世界地图现阶段更适合表达 `宏观区域 + 据点 + 遭遇分布`，不适合把黑魂式立体箱庭逐格复刻。
  - 因此本设计采用“宏观世界地图 + 据点服务 + 遭遇命名换皮 + 战斗地图氛围承接”的方案。

---

## 二、宏观分区

### 1. 中央区：余烬祭所

- 坐标区间：`x 46-55 / y 46-55`
- 定位：整张图的唯一核心安全枢纽，相当于交界后的“火之联结处”。
- 氛围：
  - 倒塌的钟楼与裂开的世界门并列。
  - 篝火、熔炉、残破王座、誓约碑混在同一片小广场中。
  - 从这里能通向西北教区、东北书库、南侧墓坡与东南断桥。
- 玩法职责：
  - 玩家初始落点。
  - 提供最完整的休整、仓储、锻火、战技和传送服务。
  - 是黑魂风格技能学习的第一站。

### 2. 西北区：炭化大教堂

- 坐标区间：`x 10-34 / y 10-34`
- 定位：主世界信仰体系与灰烬世界旧神教区碰撞后的废教权中心。
- 氛围：
  - 石柱、破碎彩窗、焚毁圣像、长阶梯和钟塔残骸。
  - 外围遍布朝圣者营火与被焚黑的墓碑。
- 玩法职责：
  - 偏 `祷告 / 誓约 / 信仰防护` 方向的服务区。
  - 适合放置高压法系敌人与祝祷型据点服务。

### 3. 东北区：沉钟书库

- 坐标区间：`x 64-88 / y 14-38`
- 定位：被潮雾淹没的禁书城片段，是异界咒术、灵魂术和禁忌档案的聚集地。
- 氛围：
  - 倾斜塔楼、泡水书架、钟声回廊、浸没中庭。
  - 光线冷、视野窄、敌人更偏法系和控场。
- 玩法职责：
  - 偏 `咒术 / 灵魂术 / 追忆` 学习区。
  - 适合提供知识解锁、法术解锁和法术熟练度训练。

### 4. 西南区：坠墓镇

- 坐标区间：`x 12-30 / y 66-84`
- 定位：埋葬坑道与旧兵营坍塌后形成的下沉聚落。
- 氛围：
  - 墓火、断墙、尸车轨、塌陷地井。
  - 地势低，通路窄，给人被困死角的感觉。
- 玩法职责：
  - 偏 `近战战技 / 黑火 / 生存补给` 的中前期练习点。
  - 适合安排强近战风格服务和墓火系技能传授。

### 5. 东南区：断桥营寨

- 坐标区间：`x 68-86 / y 70-88`
- 定位：一段被拖进交界的军桥与守备营地，连接深渊边地。
- 氛围：
  - 断桥、弩台、营火壕沟、黑铁拒马。
  - 视野开阔但退路少，是整张图最适合放军阵型遭遇的地方。
- 玩法职责：
  - 偏 `战技 / 重武器 / 站位压制` 的学习区。
  - 也可作为中后期前往深渊裂界前的最后补给站。

### 6. 东南外缘：深渊裂界

- 坐标区间：`x 82-99 / y 82-99`
- 定位：没有正式据点的高危边界区，是灰烬世界向主世界渗透最严重的位置。
- 氛围：
  - 地面像被烧穿后又被黑雾缝合。
  - 怪物命名、战斗氛围和掉落都应更“终局”。
- 玩法职责：
  - 不提供安全服务。
  - 主要作为高危野外遭遇区和后续扩展首领区。

## 三、据点设计

### 1. `embershrine_nexus` 余烬祭所

- 预设建议：`WORLD_STRONGHOLD`，`4 x 4`
- 建议坐标：`(48, 48)`
- 主视觉：
  - 断裂王座
  - 大型篝火盆
  - 熔铁砧
  - 裂界火门
- 服务：
  - `余火`
  - `骨灰储箱`
  - `锻火`
  - `战技`
  - `裂界行路`
  - `誓火部署`
- 剧情职责：
  - 玩家被卷入灰烬交界后，首先在这里恢复意识。
  - 据点中的 NPC 都知道“这里不是原本的世界”，有明显的异界错位感。

### 2. `charred_cathedral` 炭化大教堂

- 预设建议：`CAPITAL`，`3 x 3`
- 建议坐标：`(18, 18)`
- 主视觉：
  - 炭化钟楼
  - 倾倒祭坛
  - 火痕回廊
- 服务：
  - `祷告`
  - `誓约`
  - `余火`
- 剧情职责：
  - 黑魂风格“信仰/圣职”技能的主要学习点。
  - 据点语气庄严、压抑，但并非真正安全，像在衰败秩序里短暂停靠。

### 3. `drowned_archive` 沉钟书库

- 预设建议：`CITY`，`2 x 2`
- 建议坐标：`(72, 22)`
- 主视觉：
  - 倾斜书塔
  - 淹水厅堂
  - 铁钟与锁链
- 服务：
  - `咒术`
  - `追忆`
  - `流亡交易`
- 剧情职责：
  - 黑魂风格灵魂术、禁术、结晶术的学习点。
  - 追忆服务可承接知识解锁与复合技能前置。

### 4. `gravefall_town` 坠墓镇

- 预设建议：`TOWN`，`2 x 2`
- 建议坐标：`(20, 74)`
- 主视觉：
  - 墓火坑
  - 塌陷石墙
  - 黑火瓶堆栈
- 服务：
  - `余火`
  - `锻火`
  - `黑火`
- 剧情职责：
  - 更偏实战、生存、粗粝民间战技的学习点。
  - 适合传授炸壶、黑火球、破防踢击等脏招。

### 5. `broken_bridge_keep` 断桥营寨

- 预设建议：`CITY`，`2 x 2`
- 建议坐标：`(74, 74)`
- 主视觉：
  - 断桥墩
  - 壕沟营火
  - 黑铁弩架
- 服务：
  - `战技`
  - `誓火部署`
  - `流亡交易`
- 剧情职责：
  - 重武器、突进、横扫、军用站位技的学习点。
  - 是前往深渊裂界前的最后一站。

## 四、黑魂风格技能学习方案

### 设计原则

- 不直接复制黑魂原版数值与判定。
- 只保留“动作意图”和“玩法身份”：
  - 翻滚强调 `self 位移 + 闪避`
  - 盾反强调 `近战低伤 + 失衡 / 破势`
  - 灵魂箭强调 `远程单体法术`
  - 雷枪强调 `高穿透或高命中雷系突击`
  - 原力强调 `近身震退与打断`
- 奖励表现统一走当前项目的：
  - `knowledge_unlock`
  - `skill_unlock`
  - `skill_mastery`

### 推荐服务到奖励的映射

- `战技`
  - 主用 `skill_unlock + skill_mastery`
  - 少量高级招式可额外要求 `knowledge_unlock`
- `咒术`
  - 主用 `skill_unlock` 到法师/奥术技能
  - 高阶法术通过 `追忆` 先解锁知识，再由 `咒术` 服务授予
- `祷告`
  - 主用 `skill_unlock` 到牧师/圣骑士方向技能
- `追忆`
  - 主用 `knowledge_unlock`
  - 负责把“王魂、余烬、誓约、古老灵魂术残片”这些概念挂进当前成长系统

### 推荐可学技能包

1. `灰烬翻滚`
   - 黑魂原型：翻滚
   - 当前体系替换：`self / self / 消耗 stamina 1 / 冷却 1 / 位移 2 格 + 闪避 +20%`
   - 建议落点：`战技`
   - 技能来源：优先映射到通用战技或 `archer_evasive_roll` 同类模板

2. `破势盾反`
   - 黑魂原型：盾反
   - 当前体系替换：`single / 伤害 80% / 命中 +10% / 命中后施加 staggered 1`
   - 对处于防御姿态的目标追加 `armor_break`
   - 建议落点：`战技`

3. `踢击破防`
   - 黑魂原型：踢盾
   - 当前体系替换：`single / 伤害 60% / 破甲 / 移除 guard`
   - 建议落点：`战技`

4. `跳劈重斩`
   - 黑魂原型：跳劈
   - 当前体系替换：`ground -> radius 1 / 伤害 130% / 短距突进或落点攻击`
   - 建议落点：`战技`

5. `灵魂箭`
   - 黑魂原型：Soul Arrow
   - 当前体系替换：`single / range 5 / 伤害 100% / mp 1`
   - 建议落点：`咒术`
   - 可直接映射到 [mage_skills.md](D:/game/magic/docs/design/mage_skills.md) 中的 `mage_arcane_missile` 轻单体版

6. `强灵魂箭`
   - 黑魂原型：Great Soul Arrow
   - 当前体系替换：`single / range 5 / 伤害 130% / mp 2 / cooldown 1`
   - 建议落点：`咒术`
   - 可映射为 `mage_mana_burst` 或其单体奥术升级版

7. `黑火球`
   - 黑魂原型：Black Fire Orb
   - 当前体系替换：`radius 1 / 伤害 125% / mp 2 / 附加 staggered 或 灼烧`
   - 建议落点：`黑火`
   - 可映射为 `mage_fireball` 的小范围高压版本

8. `火焰壶`
   - 黑魂原型：火焰壶投掷物
   - 当前体系替换：`ground radius 1 / 伤害 90% / stamina 1 或 mp 1`
   - 建议落点：`黑火`
   - 用于给非纯法系角色提供魂系投掷物手感

9. `治愈祷告`
   - 黑魂原型：Heal
   - 当前体系替换：`single ally / heal 14 / mp 1 / cooldown 1`
   - 建议落点：`祷告`
   - 可直接映射现有 `priest_healing_light`

10. `原力震荡`
   - 黑魂原型：Force
   - 当前体系替换：`self radius 1 / 伤害 70% / staggered / 推退 1`
   - 建议落点：`祷告`
   - 可映射为圣职近身解围技

11. `雷枪`
   - 黑魂原型：Lightning Spear
   - 当前体系替换：`line 或 single / 伤害 125% / lightning / shocked`
   - 建议落点：`祷告`
   - 可映射到 `mage_thunder_lance` 一类的雷系直击技能

12. `黑暗斩痕`
   - 黑魂原型：Dark Edge / dark weapon style spell
   - 当前体系替换：`single / 伤害 110% / 奥脆 或 life_drain`
   - 建议落点：`追忆 + 咒术`
   - 适合作为沉钟书库后段的高级解锁

## 五、服务执行建议

- 当前世界预设层先落：
  - 据点名字
  - 设施名字
  - NPC 名字
  - `service_type`
  - `interaction_script_id`
- 真正的技能学习奖励，建议后续由聚落服务执行层补进：
  - `service:战技`
  - `service:咒术`
  - `service:祷告`
  - `service:追忆`
  - `service:黑火`
- 推荐 payload 形状：

```gdscript
{
  "member_id": "party_member_x",
  "source_type": "npc_teach",
  "source_label": "沉钟书库·咒术",
  "entries": [
    {
      "entry_type": "knowledge_unlock",
      "target_id": "ashen_sorcery_notes",
      "target_label": "灰烬灵魂术残卷",
      "amount": 1,
      "reason_text": "从沉钟书库抄录得到。"
    },
    {
      "entry_type": "skill_unlock",
      "target_id": "mage_arcane_missile",
      "target_label": "灵魂箭",
      "amount": 1,
      "reason_text": "已掌握黑魂风格法术的当前体系替换版。"
    }
  ]
}
```

- 这样能和当前 `CharacterManagementModule.apply_pending_character_reward(...)` 的已有处理逻辑直接对齐。

## 六、区域遭遇建议

- 当前仓库真实敌方模板只有：
  - `wolf_pack`
  - `mist_beast`
- 所以这张图的首版预设应采用“命名换皮 + 区域分布重排”：
  - `wolf_pack` 负责表现：
    - 灰墙猎犬
    - 墓火失犬
    - 深渊噬猎犬
  - `mist_beast` 负责表现：
    - 炭化朝圣者
    - 沉钟幽形
    - 裂界咒民
- 等后续敌方模板扩展后，再补：
  - 重甲骑士
  - 余烬执事
  - 深渊监视者
  - 诅咒法师
  - 教区圣职敌

## 八、首批实现边界

- 本轮只要求：
  - 这张图作为 `100 x 100` 固定预设可选
  - 据点命名和服务风格替换完成
  - 遭遇区命名和分布换成灰烬交界风格
  - 技能学习方案写进设计稿
- 本轮不强求：
  - 新敌方模板
  - 新战斗地形 profile
  - 真正的据点技能教学执行器
  - 黑魂式逐格箱庭复刻

## 九、结论

- `灰烬交界` 最适合在当前项目里作为一张 `固定生成、强主题、少量高辨识度据点` 的特殊世界预设存在。
- 黑魂风格的核心不是原封不动复制地图，而是：
  - 一个压抑但可循环的中枢据点
  - 若干风格强烈、功能不同的危险区域
  - 据点服务与技能学习形成“回到篝火，再继续前推”的节奏
- 技能学习完全可以挂进现有奖励链，不需要另造一套“魂系技能面板”系统。
