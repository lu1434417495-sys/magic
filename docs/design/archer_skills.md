# 弓箭手技能设计文档

更新日期：`2026-04-13`

说明：
- 本文档只补充设计方案，不涉及代码、资源或数值表落地。
- 命名尽量与现有仓库中的 `archer_aimed_shot`、`archer_pinning_shot`、`archer_arrow_rain`、`archer_multishot`、`archer_eagle_eye_i`、`archer_field_stride_i`、`archer_longshot_i` 保持同一套语义。
- 统一按 `射程 / 范围 / 命中 / 伤害 / 消耗 / 冷却 / 效果` 描述，便于后续挑选实现。

---

## 一、职业定位

- 核心身份：中远距离点杀、牵制、区域压制。
- 战斗节奏：通过走位重构射线，通过控制手段拖慢近战，通过多目标技能惩罚抱团。
- 资源建议：常规技能以 `stamina` 为主，少数招牌技能使用 `aura`。
- 数值倾向：弓箭手不追求最厚防御，而是通过先手、射程和站位优势换取收益。

---

## 二、复用状态

- `标记`：目标承受弓箭手单体技能时额外受到 `+10%` 伤害，持续 `2` 回合，不可叠层。
- `压制`：目标移动力 `-1`，命中 `-10%`，持续 `1` 回合。
- `断筋`：目标移动力 `-2`，且不能使用突进类位移，持续 `1` 回合。
- `预瞄`：自身下一个单体弓技命中 `+15%`，若站在高地则额外 `+10%` 伤害，持续到下一次主动攻击结束。

---

## 三、技能目录（40个）

### A. 核心输出（1-8）

1. `archer_aimed_shot` 精准射击  
   射程 `5` / 范围 `single` / 命中 `+10%` / 伤害 `130%` / 消耗 `stamina 1` / 冷却 `1`  
   稳住呼吸后射出高精度一箭；若目标本回合尚未行动，则暴击率额外 `+15%`。

2. `archer_armor_piercer` 破甲箭  
   射程 `5` / 范围 `single` / 命中 `标准` / 伤害 `110%` / 消耗 `stamina 1` / 冷却 `2`  
   命中后附加 `破甲 20%`，持续 `2` 回合，用于给后续火力开口子。

3. `archer_heartseeker` 追心箭  
   射程 `5` / 范围 `single` / 命中 `+5%` / 伤害 `125%` / 消耗 `stamina 1` / 冷却 `2`  
   若目标已带有 `标记`，本次伤害改为 `150%`；若因此击杀，则返还 `1` 点 `stamina`。

4. `archer_long_draw` 满弦狙击  
   射程 `6` / 范围 `single` / 命中 `+15%` / 伤害 `160%` / 消耗 `stamina 2` / 冷却 `3`  
   高蓄力狙杀技；若本回合已移动，则伤害降为 `135%`。

5. `archer_split_bolt` 裂风重矢  
   射程 `5` / 范围 `line` / 命中 `-5%` / 伤害 `140%` / 消耗 `stamina 2` / 冷却 `2`  
   粗重箭矢沿直线贯穿最多 `2` 个敌人，适合打站位重叠目标。

6. `archer_execution_arrow` 猎手终结  
   射程 `5` / 范围 `single` / 命中 `标准` / 伤害 `100%~180%` / 消耗 `stamina 2` / 冷却 `3`  
   根据目标已损失生命提高伤害；目标生命越低，终结收益越高。

7. `archer_double_nock` 双弦连射  
   射程 `4` / 范围 `single` / 命中 `标准` / 伤害 `70% + 70%` / 消耗 `stamina 1` / 冷却 `2`  
   连续射出两箭；第一箭未命中时不触发第二箭。

8. `archer_far_horizon` 天际远射  
   射程 `7` / 范围 `single` / 命中 `-10%` / 伤害 `150%` / 消耗 `aura 1` / 冷却 `3`  
   超远距离重箭；若自身处于 `预瞄` 或高地，则忽略命中惩罚。

### B. 机动换位（9-16）

9. `archer_skirmish_step` 游击步  
   射程 `self` / 范围 `self` / 命中 `-` / 伤害 `-` / 消耗 `stamina 1` / 冷却 `1`  
   立即移动最多 `2` 格并获得 `预瞄`，是弓箭手的基础转线手段。

10. `archer_backstep_shot` 后跃射  
   射程 `4` / 范围 `single` / 命中 `+5%` / 伤害 `95%` / 消耗 `stamina 1` / 冷却 `2`  
   先攻击再后撤 `2` 格；若后撤路径被阻挡，则只结算攻击。

11. `archer_sidewind_slide` 侧滑换位  
   射程 `4` / 范围 `single` / 命中 `标准` / 伤害 `100%` / 消耗 `stamina 1` / 冷却 `2`  
   攻击前或攻击后允许横向移动 `1` 格，用于避开直线冲锋或重构夹角。

12. `archer_running_shot` 奔袭射击  
   射程 `4` / 范围 `single` / 命中 `-5%` / 伤害 `110%` / 消耗 `stamina 1` / 冷却 `1`  
   若本回合先移动至少 `3` 格，则取消命中惩罚，并额外施加 `标记`。

13. `archer_grapple_redeploy` 索钩转位  
   射程 `4` / 范围 `self` / 命中 `-` / 伤害 `-` / 消耗 `stamina 1` / 冷却 `3`  
   牵引至高地、障碍物边缘或己方布置点附近；转位后下一个弓技射程 `+1`。

14. `archer_evasive_roll` 翻滚卸力  
   射程 `self` / 范围 `self` / 命中 `-` / 伤害 `-` / 消耗 `stamina 1` / 冷却 `2`  
   移动 `2` 格并获得直到回合结束的 `闪避 +20%`，适合脱离贴身威胁。

15. `archer_highground_claim` 抢高位  
   射程 `3` / 范围 `self` / 命中 `-` / 伤害 `-` / 消耗 `stamina 2` / 冷却 `3`  
   向指定高低差合法格位移动最多 `3` 格；若落点为高地，额外获得 `预瞄`。

16. `archer_hunter_feint` 猎步佯退  
   射程 `4` / 范围 `single` / 命中 `标准` / 伤害 `85%` / 消耗 `stamina 1` / 冷却 `2`  
   轻伤试探并附加 `标记`；命中后可免费移动 `1` 格。

### C. 控制与削弱（17-24）

17. `archer_pinning_shot` 钉射  
   射程 `4` / 范围 `single` / 命中 `标准` / 伤害 `100%` / 消耗 `stamina 1` / 冷却 `2`  
   命中后附加 `pinned`，目标下回合无法使用需要大幅移位的动作。

18. `archer_tendon_splitter` 断筋箭  
   射程 `5` / 范围 `single` / 命中 `-5%` / 伤害 `90%` / 消耗 `stamina 1` / 冷却 `2`  
   命中后施加 `断筋`；对依赖冲锋和突进的敌人压制力很强。

19. `archer_disrupting_arrow` 扰咒箭  
   射程 `5` / 范围 `single` / 命中 `+5%` / 伤害 `80%` / 消耗 `stamina 1` / 冷却 `3`  
   命中后令目标下一次技能消耗 `+1`，并使其施法类动作命中 `-15%`。

20. `archer_flash_whistle` 炫目鸣镝  
   射程 `5` / 范围 `radius 1` / 命中 `标准` / 伤害 `70%` / 消耗 `stamina 2` / 冷却 `3`  
   爆出刺耳鸣响，范围内敌人获得 `命中 -15%` 且失去反击资格，持续 `1` 回合。

21. `archer_tripwire_arrow` 绊索箭  
   射程 `4` / 范围 `line` / 命中 `标准` / 伤害 `60%` / 消耗 `stamina 1` / 冷却 `2`  
   在直线 `3` 格上布置绊索带；首个穿过的敌人立即停止移动并获得 `压制`。

22. `archer_shield_breaker` 破盾箭  
   射程 `5` / 范围 `single` / 命中 `-5%` / 伤害 `105%` / 消耗 `stamina 1` / 冷却 `2`  
   对处于格挡、护卫或防御姿态的目标追加 `+25%` 伤害，并移除其防御姿态。

23. `archer_fearsignal_shot` 惊禽哨箭  
   射程 `5` / 范围 `cone` / 命中 `标准` / 伤害 `75%` / 消耗 `stamina 2` / 冷却 `3`  
   扇形压制射击，命中敌人获得 `压制`；若目标本就带有 `标记`，额外后退 `1` 格。

24. `archer_harrier_mark` 猎印追缉  
   射程 `5` / 范围 `single` / 命中 `+10%` / 伤害 `0%` / 消耗 `stamina 1` / 冷却 `2`  
   纯功能技能，给目标施加强化版 `标记` `2` 回合；所有友军远程攻击对此目标命中 `+10%`。

### D. 区域压制与多目标（25-32）

25. `archer_multishot` 连珠箭  
   射程 `5` / 范围 `multi_unit` / 命中 `-5%` / 伤害 `85%` / 消耗 `stamina 2` / 冷却 `2`  
   一次选择 `2-3` 个不同敌方单位依次射击；同一目标不可重复选择。

26. `archer_arrow_rain` 箭雨  
   射程 `5` / 范围 `radius 1` / 命中 `-10%` / 伤害 `80%` / 消耗 `stamina 2` / 冷却 `3`  
   标准范围火力；命中的敌人额外获得 `压制`，适合拆散抱团站位。

27. `archer_fan_volley` 扇幕齐射  
   射程 `4` / 范围 `cone` / 命中 `标准` / 伤害 `90%` / 消耗 `stamina 2` / 冷却 `2`  
   中距离扇面清线技，对近距离多前排组合威胁较高。

28. `archer_suppressive_fire` 压制射击  
   射程 `5` / 范围 `line` / 命中 `标准` / 伤害 `70%` / 消耗 `stamina 2` / 冷却 `3`  
   对一条直线区域持续覆盖；被命中的敌人获得 `压制`，该路径在下回合前移动成本 `+1`。

29. `archer_breach_barrage` 贯阵齐发  
   射程 `6` / 范围 `line` / 命中 `-10%` / 伤害 `120%` / 消耗 `stamina 2 + aura 1` / 冷却 `4`  
   高阶穿阵技，直线上的每命中一个敌人，下一名目标额外承受 `+10%` 伤害。

30. `archer_blast_arrow` 爆裂箭  
   射程 `5` / 范围 `radius 1` / 命中 `标准` / 伤害 `95%` / 消耗 `aura 1` / 冷却 `3`  
   在落点爆开冲击；对轻型掩体、召唤物或低护甲后排有较高压制价值。

31. `archer_hunting_grid` 狩猎网阵  
   射程 `4` / 范围 `radius 2` / 命中 `标准` / 伤害 `60%` / 消耗 `aura 1` / 冷却 `4`  
   在区域内撒布索绳与短箭；敌人首次离开区域时再受一次 `50%` 伤害并获得 `断筋`。

32. `archer_killing_field` 猎场封锁  
   射程 `5` / 范围 `radius 2` / 命中 `-10%` / 伤害 `70%` / 消耗 `aura 2` / 冷却 `5`  
   招牌领域技；区域持续 `2` 回合，敌人每次在其中结束行动时受到追击伤害并自动附加 `标记`。

### E. 被动与专精（33-40）

33. `archer_eagle_eye_i` 鹰眼  
   类型 `passive`  
   提升观察与出手稳定性；建议保留当前方向，额外补充“每回合第一次单体弓技命中 `+10%`”。

34. `archer_field_stride_i` 战地步幅  
   类型 `passive`  
   强化射后重定位；每回合首次使用弓技后，获得 `1` 格免费位移。

35. `archer_longshot_i` 远射校准  
   类型 `passive`  
   强化远距离稳定性；对 `5` 格及以上的单体射击忽略前 `10%` 命中惩罚。

36. `archer_quick_nock` 快速搭箭  
   类型 `passive`  
   若本回合未移动，则第一个弓技 `stamina` 消耗 `-1`；最低不低于 `1`。

37. `archer_low_profile` 低姿隐步  
   类型 `passive`  
   当自身周围没有相邻敌人时，闪避 `+15%`；在草丛、灌木、矮墙边移动时再额外 `+1` 格有效位移。

38. `archer_killzone` 杀伤扇区  
   类型 `passive`  
   对带有 `标记` 或 `压制` 的目标造成伤害 `+10%`，鼓励先控后杀的循环。

39. `archer_predator_patience` 猎手耐心  
   类型 `passive`  
   连续两次攻击同一目标时，第二次命中 `+5%`、伤害 `+10%`；切换目标后重置。

40. `archer_last_arrow` 终矢直觉  
   类型 `passive`  
   每场战斗首次将敌人打到 `30%` 生命以下时，自身立即获得一次不消耗 `stamina` 的 `精准射击` 机会。

## 四、设计原则

- 弓箭手的强度应建立在“先手与射线管理”上，而不是硬吃伤害。
- 单体点杀技与多目标压制技应严格区分，不让一个技能同时吃满两类收益。
- 机动技能以“打完换位”为主，不建议给过多无条件穿模或长距离位移。
- 控制技能的价值应更多体现在压慢敌人、拆掉阵型，而不是直接替代硬控职业。
- `aura` 技能应是高价值决策，不应成为每回合常规动作。

- `archer_multishot`
- `archer_armor_piercer`

- `archer_skirmish_step`
- `archer_disrupting_arrow`

- `archer_field_stride_i`

这套首批池已经覆盖点杀、控场、转位、AOE、被动成长五条主线，能较快形成完整职业手感。
