## 一、统一规则

### 技能字段
- name：技能名
- range：射程
- area：范围（single / line / cone / radius / self）
- hit：命中率
- damage：伤害倍率
- cost：资源消耗（stamina / aura）
- cooldown：冷却
- effects：效果列表
- duration：持续回合

## 三、AI逻辑

### 优先级
1. 击杀目标
2. 控制高威胁单位
3. 保命（回血/防御）
4. 位移接敌
5. AOE最大收益

---

## 四、目标评分模型

目标评分 =
击杀价值 +
威胁值 +
命中收益 +
地形收益 -
风险

---

## 五、平衡规则

- 控制递减
- 位移限制（CD ≥2）
- 斗气不可滥用
- 连击最多2次追击
